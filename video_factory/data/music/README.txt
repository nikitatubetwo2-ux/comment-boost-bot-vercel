ФОНОВАЯ МУЗЫКА ДЛЯ ВИДЕО
========================

Положите сюда MP3 файлы с фоновой музыкой.

Называйте файлы по настроению:
- epic_dramatic.mp3 — для военных/исторических видео
- mysterious_ambient.mp3 — для загадок/тайн
- emotional_orchestral.mp3 — для драматичных историй
- dark_tense.mp3 — для напряжённых моментов
- cinematic.mp3 — универсальная кинематографичная

Программа автоматически подберёт музыку по настроению проекта.

ВАЖНО: Используйте только royalty-free музыку!

Рекомендуемые источники:
- YouTube Audio Library (бесплатно)
- Pixabay Music (бесплатно)
- Epidemic Sound (платно, но качественно)
- Artlist (платно)
